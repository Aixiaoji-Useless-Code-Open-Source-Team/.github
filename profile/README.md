# Aixiaoji-Useless-Code-Open-Source-Team

## 简介
Aixiaoji-Useless-Code-Open-Source-Team 是一个追求滑稽和娱乐的开源团队。

## 项目介绍
该团队致力于创造全是无用的、滑稽的、屎山代码的世界（bushi）。其中，他们的项目“Uselelan（UL）解释器”是一个滑稽、毫无用处的编程语言解释器，它拥有荒谬离奇的语法规则，让编程体验变得有趣而滑稽。
